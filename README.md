# AI & ML Engineer Portfolio

A modern, responsive portfolio website showcasing Artificial Intelligence, Machine Learning, and Full Stack Development projects.

## Project Info

**Developer**: Mohammed Sameer
**Role**: AI & ML Engineer
**Tech Stack**: React, TypeScript, Tailwind CSS, Framer Motion, Vite

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Install dependencies**
    ```sh
    npm install
    ```

3.  **Start the development server**
    ```sh
    npm run dev
    ```

## Features

- **Responsive Design**: Built with Tailwind CSS for all device sizes.
- **Animations**: Smooth transitions using Framer Motion.
- **Contact Form**: Integrated with EmailJS for real-time messaging.
- **Project Showcase**: Highlighted AI/ML and Web Development projects.
